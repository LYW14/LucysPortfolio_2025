# Lucy's Portfolio Website

A creative, interactive personal portfolio built as an explorable kitchen scene. Click around the kitchen to discover projects, hobbies, travel experiences, and more.

🌐 **Live Site:** https://lucys-portfolio-2025.vercel.app

---

## About

Hi, I'm Lucy — a data scientist with experience spanning full-stack development, machine learning, and cognitive/biological research.

This site is my creative way of sharing the projects I've worked on and the things I care about. Rather than a traditional portfolio, I designed an interactive kitchen environment where visitors can explore different aspects of my work and interests by clicking on objects throughout the scene.

---

## Tech Stack

* **Framework:** React 18 + TypeScript
* **Build Tool:** Vite
* **Styling:** Tailwind CSS + custom CSS
* **Deployment:** Vercel

---

## Running Locally

Clone the repository and start the development server:

```bash
git clone https://github.com/LYW14/LucysPortfolio_2025.git
cd LucysPortfolio_2025
npm install
npm run dev
```

Then open:

```text
http://localhost:5173
```
